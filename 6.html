4.0 out of 5 stars Great little inexpensive, expandable, laptop.
By Gavin Scott on October 5, 2016
Vine Customer Review of Free Product ( What's this? )
For the price this is a very nice laptop with the brand new 7th gen Intel CPU which also has reasonable DirectX 3D performance for gaming. It's nicely expandable (up to 32GB of memory and an easily accessible 2.5" hard drive which could be swapped out with an SSD for performance).

I replaced the single 4GB memory DIMM that it came with with this: Crucial 32GB Kit (16GBx2) DDR4 2133 MT/s (PC4-17000) SODIMM 260-Pin Memory - CT2K16G4SFD8213 to max out the memory at 32GB. At some point I may swap out the horribly slow 5400 RPM laptop drive (pretty much the same thing you get in all lower priced laptops these days) for an SSD, but with the extra memory performance is much improved and switching between applications that are already loaded is always instantaneous.

One kind of odd thing is that it comes with a built-in CD/DVD drive. Not something you see much in these days of "everything is a download". If you want to burn archival backups, or your car still has a CD player, this might be a useful feature for you.

Gaming performance is probably fine for a lot of purposes and if you're willing to run on lower graphics settings then you should be able to run most games at reasonable frame-rates. It will run the latest World of Warcraft at graphics level 4 at 30-60fps at the full native 1920x1080 display resolution just as an example. Anything more casual than that will be great, and things more intense may need to be on low graphics settings.

The illuminated keyboard's LEDs for the top row of keys kind of shine in your face at lower angles which is a bit annoying. Read more ›

4.0 out of 5 stars Good cheap laptop
By Bradford B. on October 22, 2016
Verified Purchase
Nice laptop for cheap, with a few cons.

Pros:

- Nice 1080p screen, anti-glare coating isn't too heavy. Blue filter software helps with eye strain.
- Fast Kaby Lake i3 for the price.
- User upgrade able RAM and HDD slots, no soldered stuff like many cheap laptops.
- Overall build quality is quite nice, screen doesn't creak at all and the hinge is suitably stiff.
- Very little pre-installed software. Zero bloat.

Cons:
- Touchpad was giving me all sorts of problems, jumping around the screen and acting as if I was using "gestures". I ended up disabling advanced touchpad functions in the BIOS and this solved the problems, now the touchpad is actually useful.
- HDD is incredibly slow. You might as well watch mold grow in a petri dish. Replaced instantly for a spare 250GB SSD I had and it makes the system 1 billion percent better.

Overall a solid little rig, I replaced the single 4GB DDR4 2400 Kingston stick with 2x8GB of Samsung DDR4 2133 I had on hand and it works great. Not really worried about the RAM speed on an internet only machine. Screen is plenty big and internet performance is excellent. I bought this particular laptop specifically knowing I was going to use RAM and an SSD I already had, but I would definitely suggest at least a cheap SSD if you're looking at this model for optimum results, it makes the system so much more responsive over the dog of a Seagate hard drive it ships with.

**NOTE - The machine WILL NOT turn on under battery power if you have the access cover off! It has a little microswitch to prevent that, it has to be plugged in to turn on if you want to test some new RAM or something before you close it up.
